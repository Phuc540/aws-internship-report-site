This section lists and introduces the three blog posts completed during our internship. These blogs helped us summarize technical knowledge, organize information clearly, and connect what we learned with practical cloud computing, AWS services, and the GreenLens Kids project.

### [Blog 1 - AWS Cloud Fundamentals and Practical Learning](3.1-Blog1/)

This blog introduces fundamental knowledge about AWS Cloud and explains how cloud computing supports modern application development. Through this blog, we learned more about important AWS concepts such as cloud infrastructure, compute, storage, networking, security, and monitoring. The blog also helped us understand how AWS services can be applied in real-world projects and how cloud knowledge supports the development process during our internship.

### [Blog 2 - Applying AWS and AI in Real-world Projects](3.2-Blog2/)

This blog focuses on how AWS and AI-related technologies can be applied to practical project scenarios. It discusses the role of cloud services in supporting scalability, security, data processing, and intelligent application features. Through this blog, we gained a clearer understanding of how AI and cloud computing can work together to build useful solutions, especially in projects like GreenLens Kids that involve AI-based features and user interaction.

### [Blog 3 - Frontend Development, API Integration, and Project Deployment](3.3-Blog3/)

This blog presents the process of working with frontend development, API integration, and project deployment during the internship. It introduces how user interfaces are designed, how frontend components interact with backend APIs, and how deployment helps make the project accessible online. Through this blog, we improved our understanding of UI/UX, API flow, error handling, testing, and the importance of connecting technical implementation with user experience.
