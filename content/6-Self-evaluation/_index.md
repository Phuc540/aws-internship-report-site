During my internship at Amazon Web Services (AWS) Vietnam from 17/04/2026 to 30/07/2026, I had the opportunity to study and work in a professional, dynamic, and clearly oriented environment. This period helped me gain practical exposure to AWS Cloud knowledge, teamwork processes, product development, and how to apply technology to a specific project.

In the **GreenLens Kids** project, I took on the role of **Frontend Developer**. I also participated in the process of designing interfaces, adjusting UI/UX, and integrating APIs for user-facing features. My main responsibilities included designing and building interfaces in Figma, developing the character creation screen for children, integrating the child profile creation API, building the AI Camera result interface, integrating mascot voice guidance, designing the Daily Streak interface, integrating the activity streak API, and adjusting the UI/UX for the Admin Dashboard.

Through the assigned tasks, I had the opportunity to improve my frontend development skills, interface organization, user state handling, data validation, API calls, loading/error/success response handling, client-side data storage, and feature flow testing. In addition, I also learned how to design interfaces suitable for children, ensuring that the interface is friendly, easy to use, intuitive, and highly interactive.

Regarding my working attitude, I always tried to complete assigned tasks on schedule, proactively checked for bugs, adjusted the interface based on feedback, and coordinated with team members to ensure that features worked stably. During the working process, I realized that frontend development does not only require coding ability, but also requires careful attention to user experience, understanding of system flow, communication with the backend side, and thorough checking of possible error cases.

Besides project tasks, I also participated in learning activities through AWS Skill Builder labs, knowledge-sharing sessions, and community events. These activities helped me expand my understanding of AWS, DevOps, AI, cloud architecture, as well as career orientation in the information technology field.
